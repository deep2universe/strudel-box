# Strudel Box

VS Code extension that embeds a Strudel live coding environment for creating music patterns.

## Core Features
- Webview-based REPL for writing and evaluating Strudel patterns
- CodeMirror 6 editor with JavaScript syntax highlighting
- Play/Stop controls with keyboard shortcuts (Ctrl/Cmd+Enter to play, Ctrl/Cmd+. to stop)
- Theme support (Default/Cyberpunk, Halloween, 8-Bit)
- Load/save .strudel pattern files
